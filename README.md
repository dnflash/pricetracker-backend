# pricetracker-backend

## How To Run
- Use Go 1.18+

On the project root directory run:

```
go run cmd/pricetracker.go
```
Or compile it first with:

```
go build cmd/pricetracker.go
```
and then execute the generated main binary.

On Windows:
```
./pricetracker.exe
```
On Linux/Mac:
```
./pricetracker
```
